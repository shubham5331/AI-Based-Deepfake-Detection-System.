# AI-Based Deepfake Detection System

## Overview
The AI-Based Deepfake Detection System is a web application that detects whether an uploaded image or video is real or manipulated. The system uses deep learning and computer vision techniques to identify deepfake content.

Users can upload media files through a simple web interface, and the system analyzes facial patterns and inconsistencies to determine if the content is authentic or fake.

## Features
- Upload images for deepfake detection
- AI-based analysis using deep learning
- Real-time detection results
- User-friendly web interface
- Drag and drop file upload
- Instant prediction output

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Backend
- Python
- Flask API

### AI / Machine Learning
- TensorFlow.js
- OpenCV
- Deep Learning Model

## How It Works
1. User uploads an image or video.
2. The frontend sends the file to the Flask backend.
3. The backend processes the file using OpenCV.
4. The deep learning model analyzes the image.
5. The system returns the result as **Real** or **Deepfake**.
